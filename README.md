# 📊 Sales & Profit Analysis Dashboard

## 📌 Project Overview
This project is an interactive **Sales Dashboard** built using Microsoft Excel to analyze sales performance, profit trends, and customer insights.

The dashboard helps in understanding business performance through visualizations and KPIs.

---

## 🎯 Objectives
- Analyze total sales and profit
- Identify top-performing categories and customers
- Track monthly sales trends
- Understand regional (state-wise) performance

---

## 🛠 Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

---

## 📊 Dashboard Features
- 📈 Total Sales and Total Profit KPIs
- 📅 Monthly Sales Trend Analysis
- 📦 Category-wise Sales Performance
- 🧑 Top 5 Customers by Profit
- 🗺 State-wise Sales Distribution (Map)
- 🎛 Interactive Filters (Category, Year, Month)

---

## 📈 Key Insights
- Highest sales recorded in **December (~$241K)**
- **Technology category** contributes the highest profit
- **California** is the top-performing state
- A few customers contribute significantly to overall profit
